# EcoLuxe — Refillable Deodorant Website (D2C Starter)

This folder contains a ready-to-deploy static website for a D2C eco-luxury refillable deodorant brand named *EcoLuxe*.

## What is included
- index.html — Homepage with hero, product highlight, subscription form
- product.html — Product detail and purchase form placeholder
- about.html — Brand story and sourcing
- css/styles.css — High-quality responsive styles (minimalist luxury)
- js/script.js — Minimal JavaScript for forms and basic UX (placeholder for real integrations)
- images/* — Logo and product mockup SVG assets

## How to use
1. Unzip and review files.
2. For local testing, open `index.html` in your browser.
3. To host on GitHub Pages:
   - Create a new GitHub repository.
   - Upload these files to the repo root (or `gh-pages` branch).
   - In repo settings, enable GitHub Pages from the `main` branch -> `/ (root)`.
4. To add real checkout/subscriptions, integrate with:
   - Shopify (recommended for scale) — or
   - Stripe + serverless backend for custom checkout
   - Recharge or Bold for subscription management on Shopify

## Design & Psychology notes
- Minimalist typography and high contrast CTA to position premium brand.
- Subscription flow emphasized to boost customer lifetime value.
- Product-centric hero to drive conversion (one hero product strategy).
